Senior Frontend Engineer

Engineering Full-time Senior Level

Overview

TechFlow Inc. is seeking a Senior Frontend Engineer to lead the development of our next-generation enterprise workflow tools. In this role, you will architect complex user interfaces, optimize application performance, and mentor junior developers.

Responsibilities

  • Architect and build scalable, robust frontend applications using React and TypeScript.
  • Collaborate with design teams to implement complex, responsive UI designs utilizing Tailwind CSS.
  • Optimize application performance for maximum speed and scalability.
  • Mentor mid-level and junior engineers, conducting code reviews and fostering best practices.

Requirements

  • 5+ years of professional experience in frontend development.
  • Expert-level proficiency in React, TypeScript, and modern JavaScript.
  • Strong experience with utility-first CSS frameworks like Tailwind CSS.
  • Deep understanding of web performance optimization techniques.

Benefits

  • Competitive base salary + equity options.
  • Comprehensive health, dental, and vision insurance.
  • Flexible hybrid work model and unlimited PTO.
